Settings this script changes:

Sets terminal IP address.

Sets terminal NetBIOS name.

Sets TERM.$$$

Copies relevant INI files from another terminal:
	AMS32.INI
	DELIVERY.INI
	ORDRSCRN.INI
	WINTERM.INI

Generates POSIW.INI using the following information:
	Device number (from spcwin.ini)
	Terminal type (redundant, normal)

Installs and starts UltraVNC with correct passwords.
